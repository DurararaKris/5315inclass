What is Docker

Docker is a containerization platform. A container platform is a environment that you can package, modify and distribute your application to anywhere that can run docker. It makes sure that anywhere you want to run your application has the same environment as the one that you develop your application. You can create isolated environment for each app running on your computer, and improve security and efficiency. 

source: https://docs.docker.com/engine/docker-overview/

What is Nginx

Nginx is a HTTP server and reverse proxy. It can provide wide covered range of services, and is very known for its high performance, stability, rich feature set, simple configuration and low resource consumption. It was first developed by Russian programmer Igor Sysoev and now transferred to the Nginx company founded in 2012. Now Nginx become a platform that can provide many kinds of online services such as Load balancers, Microservices and API gateways.

Source: https://www.nginx.com, https://www.nginx.com/resources/wiki/

What is VM (virtual machine)

Virtual machine is generally a emulation of a computer system. It is usually running  under a given system environment that to create a virtual computer which can provide the same functionality as a real computer, but does not require the hardwares. Virtual machine are specialized into system virtual machines and process virtual machines, in which that one can provide the full emulation of a computer and the other can only provide a executable platform for certain programs.

Source: https://en.wikipedia.org/wiki/Virtual_machine

Who is Cloud Computing Service Provider - Digitalocean

The virtual machine provider that we used in class is DigialOcean. Digital Ocean is a industry leading virtual machine and cloud solution provider. It allows you to customize your machine’s system aka. mirror, size of hardware usage, and also the location of virtual machine origin.
